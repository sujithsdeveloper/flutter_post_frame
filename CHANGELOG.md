## Changelog

## 0.0.2
* Added `PostFrameScrollBuilder` - A widget wrapper that provides post-frame callbacks with widget size and scroll controller management
* Added `FlutterPostFrameScrollMixin` - A mixin for managing multiple scroll controllers with post-frame callbacks
* Added `PostFrameScrollManager` - A utility class for creating and managing multiple named scroll controllers


## 0.0.1

* Initial release
* Added `FlutterPostFrameMixin` - A mixin to execute callbacks after the first frame
* Added `PostFrameBuilder` - A widget wrapper that provides post-frame callbacks with widget size
